Ajuda & Contato

EletroNIC

Versão 1.0 (KMZ Release)

1. Simulador Básico
O Simulador permite testar o comportamento de portas lógicas individuais.

1. Selecione a porta (ex: AND, OR).
2. Toque nos interruptores (Switches) para mudar as entradas entre 0 e 1.
3. Observe o LED e o diagrama mudarem de cor (Verde = Ligado/1, Cinza = Desligado/0).
2. Tabela Verdade & Expressões
Gere tabelas verdade completas para qualquer lógica.

Modo Expressão Livre: Use o teclado especial para digitar equações como (A+B)*C ou A ⊕ B. O app gera todas as combinações de entrada (000 a 111) e mostra a saúde.

Dica: Você pode enviar a expressão direto para o Mapa de Karnaugh ou ver o Circuito.
3. Mapa de Karnaugh (K-Map)
Ferramenta para simplificar lógica booleana.

1. Escolha o número de variáveis (2, 3 ou 4).
2. Toque nas células para alternar entre 0, 1 e X (Don't Care).
3. Clique em Resolver Mapa. O app encontrará a equação mais simples possível, priorizando grupos maiores (Oitavas > Quadras > Pares).
4. Interatividade: Toque nos termos da equação resultante para ver quais células do mapa foram agrupadas!
4. Desenhista de Circuitos
Cria automaticamente o esquema elétrico (diagrama) a partir de uma expressão.

- Suporta portas AND, OR, NOT, XOR, XNOR.
- Desenha em "Cascata": Se houver muitas entradas, ele conecta portas em sequência de forma organizada.
- Linhas retas e organizadas (padrão industrial).
- Exportar: Use o botão "Visualizar p/ Print" para expandir o circuito e tirar um print da tela.
5. Calculadora de Bases
Realize conversões e contas entre bases numéricas.

- Digite em qualquer campo (DEC, BIN, OCT, HEX) para converter instantaneamente.
- Use a calculadora abaixo para somar, subtrair, multiplicar ou dividir números de bases diferentes (ex: Binário + Hexadecimal).
